# 脚本目录

包含各种辅助脚本。

## 脚本列表

| 脚本 | 说明 | 使用方法 |
|------|------|----------|
| `init_db.sh` | 数据库初始化 | `./init_db.sh` |
| `backup_db.sh` | 数据库备份 | `./backup_db.sh` |
| `dev_start.sh` | 启动开发环境 | `./dev_start.sh` |
| `deploy.sh` | 生产环境部署 | `./deploy.sh` |

## 权限设置

```bash
# 给脚本添加执行权限
chmod +x scripts/*.sh
```

## 注意事项

1. 执行数据库脚本前请确保已备份重要数据
2. 生产环境执行脚本前请先在测试环境验证
3. 脚本中的默认密码仅用于开发环境，生产环境必须修改
